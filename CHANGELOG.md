# Changelog

## 0.1.0 - 2026-04-20

- Added the initial Uterm release.
- Implemented the fixed-header UDP protocol and stop-and-wait ARQ.
- Added multi-client session isolation and 5-second heartbeat handling.
- Added a modern CustomTkinter desktop client.
- Added ANSI terminal rendering via `pyte`.
- Added POSIX PTY support for advanced terminal applications.
- Added automated unit and end-to-end validation.
